# Kingdom Come: Deliverance II - Codex EPUBs

Illustrated EPUB editions of the Kingdom Come: Deliverance II in-game codex, available in multiple languages.

Each book contains:

- 272 codex entries across all six in-game sections
- embedded article artwork and cover image
- internal links between included codex entries
- story spoiler markers on the small number of entries that need them
